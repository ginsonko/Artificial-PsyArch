## 项目简介

Artificial-PsyArch 是一个基于谐振认知假说设计的类脑认知系统原型。它将人类认知过程中的电信号、能量共鸣、全息记忆等概念映射为可计算的工程模块，旨在探索一种能够持续运行、具备联想、情感与自适应能力的 "人工心智" 架构。

本项目代码与同名论文同步发布，所有内容均遵循严格的非商业学术研究许可。欢迎学术界同仁和研究人员参考、验证并提出改进建议。

## 背景与动机

当前人工智能在模式识别、语言生成等任务上取得了显著成就，但在真正的理解、动态联想、情感体验等方面仍与人类心智存在本质差距。我们尝试从物理第一性原理出发，重新思考智能的本质：

- **谐振认知假说**将大脑视为一个由神经电信号驱动的共鸣系统

- 记忆以全息形式分布，思维是不同频率信号的干涉与传播

- 该假说为解释直觉、推理、情感与认知的耦合提供了一种统一的物理隐喻

本项目旨在用软件工程方法验证这一假说的可行性，并逐步构建一个可运行的认知架构原型。

## 核心特性

### 🔍 全息深度数据库

采用 "波峰索引" 与 "贪心匹配" 机制存储和提取记忆，支持时序与共现信息的动态组合，实现自然的概念泛化与联想。

### ⚡ 实虚能量与认知熵

将外界刺激（实能量）与内部预期（虚能量）统一建模，以 "认知熵" 度量预期与现实的差异，驱动注意力分配与学习。

### 🎯 注意力滤波器

模拟人类注意力的聚焦、滤波与疲劳现象，根据认知熵动态调节内源性刺激的构成，保证有限计算资源的高效利用。

### 💫 情绪与认知感受模块

内置六大神经递质（多巴胺、肾上腺素、催产素、血清素、内啡肽、皮质醇）模型，以及 "正确感 / 违和感"" 惊喜 / 惊吓 " 等数十种认知感受，使系统能够产生类人的情感反应。

### 🧠 长期心理动力缓存

通过 "认知感受长期缓存表" 与 "行为意图长期缓存表" 保存重大认知事件，使其能在后续认知过程中被重新唤醒，模拟人类的长期情绪记忆。

### 🔄 闭环认知滴答

每轮认知循环包含状态衰减、注意力滤波、内外刺激融合、数据库匹配、情绪评估、行动触发等步骤，构成一个完整的思维迭代单元。

## 快速运行

本仓库已保留当前原型默认配置、先天规则、自适应调参规则、6 个示例数据集，以及一条 300 tick 演示运行记录。拉取后通常只需要安装依赖并启动观测台。

1. 双击 `依赖自检与安装.bat` 安装/检查 Python 依赖。
2. 双击 `快速启动观测台.bat` 启动后端。
3. 等控制台出现 `AP Observatory Web UI: http://127.0.0.1:8765/next/` 后，在浏览器打开 `http://127.0.0.1:8765/next/`。

如果目录里积累了大量临时输出，可双击 `清理日志和临时输出.bat`。这个工具只清理日志、临时探针、缓存和普通运行输出，不会清理源码、数据集配置、先天规则、主 HDB 数据或长期实验运行记录。

## 常用功能

- **实时观测**：在观测台首页输入一句话运行单 tick，可以查看文本感受、状态池、HDB 查存、感应赋能、认知感受、行动等流程细节。
- **长期实验**：进入“长期实验”页面，选择数据集并运行 300/3000/30000 tick 等规模化实验；可在实时监控、指标图表、运行记录中观察状态池 Top、ER/EV、认知压、行动和性能变化。
- **内置数据集**：当前发布保留 6 个数据集，分别是 `companion_bot_chat_pure_small_v1 / medium_v1 / large_v1` 和 `ap_behavioral_curriculum_small_v2 / medium_v2 / large_v2`。
- **默认演示记录**：仓库包含 `exp_ap_behavioral_curriculum_small_v2_20260506_004306_7604` 这条 300 tick 运行记录。初次打开长期实验页面时，即使还没有重新运行，也可以直接查看图表和对应的大模型审查报告。
- **LLM 审查**：长期实验页可以调用大模型分析运行记录。此功能需要在页面中配置自己的 API URL、API Key 和模型名；仓库不会包含作者的私有 URL 或 Key。
- **自适应调参器**：默认保留当前调参规则和持久化参数，可直接使用。调参器的 LLM 辅助分析也需要单独配置自己的 API URL 和 Key。

当前默认配置使用新版 growth 主链，默认关闭历史结构级查存一体和认知拼接对照路径，适合直接复现当前原型的主线运行效果。
