## 项目简介

Artificial-PsyArch 是一个基于谐振认知假说设计的类脑认知系统原型。它将人类认知过程中的电信号、能量共鸣、全息记忆等概念映射为可计算的工程模块，旨在探索一种能够持续运行、具备联想、情感与自适应能力的 "人工心智" 架构。

本项目代码与同名论文同步发布，所有内容均遵循严格的非商业学术研究许可。欢迎学术界同仁和研究人员参考、验证并提出改进建议。

## 背景与动机

当前人工智能在模式识别、语言生成等任务上取得了显著成就，但在真正的理解、动态联想、情感体验等方面仍与人类心智存在本质差距。我们尝试从物理第一性原理出发，重新思考智能的本质：

- **谐振认知假说**将大脑视为一个由神经电信号驱动的共鸣系统

- 记忆以全息形式分布，思维是不同频率信号的干涉与传播

- 该假说为解释直觉、推理、情感与认知的耦合提供了一种统一的物理隐喻

本项目旨在用软件工程方法验证这一假说的可行性，并逐步构建一个可运行的认知架构原型。

## 核心特性

### 🔍 全息深度数据库

采用 "波峰索引" 与 "贪心匹配" 机制存储和提取记忆，支持时序与共现信息的动态组合，实现自然的概念泛化与联想。

### ⚡ 实虚能量与认知熵

将外界刺激（实能量）与内部预期（虚能量）统一建模，以 "认知熵" 度量预期与现实的差异，驱动注意力分配与学习。

### 🎯 注意力滤波器

模拟人类注意力的聚焦、滤波与疲劳现象，根据认知熵动态调节内源性刺激的构成，保证有限计算资源的高效利用。

### 💫 情绪与认知感受模块

内置六大神经递质（多巴胺、肾上腺素、催产素、血清素、内啡肽、皮质醇）模型，以及 "正确感 / 违和感"" 惊喜 / 惊吓 " 等数十种认知感受，使系统能够产生类人的情感反应。

### 🧠 长期心理动力缓存

通过 "认知感受长期缓存表" 与 "行为意图长期缓存表" 保存重大认知事件，使其能在后续认知过程中被重新唤醒，模拟人类的长期情绪记忆。

### 🔄 闭环认知滴答

每轮认知循环包含状态衰减、注意力滤波、内外刺激融合、数据库匹配、情绪评估、行动触发等步骤，构成一个完整的思维迭代单元。
